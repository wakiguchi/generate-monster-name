# Generate-Monster-Name

Predict monster name based on Pokemon

## Description

1. Train  
Input: Pokemon status  
Output: Pokemon name  
2. Generate  
Input: Random status  
Output: New monster name  

## Features

- Deep learning
- Autoregression

## Requirement

- Python >= 3.6.3
- Tensorflow >= 1.3.0
- pandas
- numpy
- sklearn

## Usage

1. Training name model.

    python train_name.py

2. Generate name  

    python test_name.py
